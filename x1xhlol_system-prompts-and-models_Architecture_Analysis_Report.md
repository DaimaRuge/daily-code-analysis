## 一、项目概述

### 1.1 核心定位

system-prompts-and-models-of-ai-tools 是一个 **AI 工具系统提示词和模型的合集仓库**：

收录了各种 AI 编程工具的 system prompts，让开发者可以了解、比较和借鉴这些工具的设计思路。

### 1.2 解决的问题

AI 工具的黑盒问题：
- **不知道工具怎么想**：看不到 AI 工具的系统提示
- **无法借鉴设计**：不知道好的 Agent prompt 怎么写
- **安全风险**：系统提示暴露可能导致 prompt 注入攻击

### 1.3 收录的 AI 工具

| 类别 | 工具 |
|------|------|
| **AI 编程** | Augment Code, Claude Code, Cluely, CodeBuddy, Comet, Cursor, Devin AI, Junie, Kiro, Leap.new |
| **Agent** | Claude, GPT, Gemini 等 |
| **其他** | 各类 AI 助力和 Agent 工具 |
